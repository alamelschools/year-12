## **Semester 1: Introduction to Java and Object-Oriented Programming (OOP)**

### **Overview:**
- **Duration:** 15 weeks (4 months)
- **Frequency:** 1 class per week
- **Focus:** Java programming fundamentals, OOP concepts, and practical application development.
- **Assessments:** Monthly exams and a final exam covering all semester content.

---

### **Month 1: Foundations of Java**

#### **Week 1: Introduction to Java and Development Environment**
- **Content:**
  - Overview of Java and its applications.
  - Installing and setting up JDK and an IDE (e.g., IntelliJ IDEA or Eclipse).
  - Writing and running a simple "Hello, World!" program.
- **Additional Tasks:**
  - Customize the IDE with preferred settings.
  - Explore basic IDE features like code completion and debugging tools.

#### **Week 2: Data Types, Variables, and Basic Operators**
- **Content:**
  - Primitive data types (`int`, `double`, `char`, `boolean`).
  - Declaring, initializing, and using variables.
  - Basic operators: arithmetic, relational, and logical.

#### **Week 3: Control Flow Statements**
- **Content:**
  - Conditional statements (`if`, `else if`, `else`, `switch`).
  - Looping constructs (`for`, `while`, `do-while`).

#### **Week 4: Arrays and Basic Input/Output**
- **Content:**
  - Introduction to arrays: declaring, initializing, and accessing elements.
  - Basic input using `Scanner` and output using `System.out`.

#### **End of Month 1: **Monthly Exam**
- **Exam Components:**
  - **Multiple-Choice Questions:** Covering data types, variables, operators, control flow, and arrays.
  - **Short Answer Questions:** Explain the difference between `while` and `do-while` loops.

---

### **Month 2: Introduction to OOP Concepts**

#### **Week 5: Methods and Functions**
- **Content:**
  - Defining and invoking methods.
  - Method parameters and return types.
  - Method overloading.

#### **Week 6: Classes and Objects**
- **Content:**
  - Understanding classes and objects.
  - Defining classes with attributes and methods.
  - The `this` keyword.

#### **Week 7: Encapsulation and Access Modifiers**
- **Content:**
  - Principles of encapsulation.
  - Access modifiers: `private`, `protected`, `public`.
  - Getters and setters.

#### **Week 8: Constructors and Object Initialization**
- **Content:**
  - Types of constructors: default and parameterized.
  - Constructor overloading.
  - Object creation using the `new` keyword.

#### **End of Month 2: **Monthly Exam**

---

### **Month 3: Advanced OOP Concepts**

#### **Week 9: Inheritance and Polymorphism**
- **Content:**
  - Understanding inheritance and the `extends` keyword.
  - Method overriding.
  - Polymorphism: dynamic method dispatch.

#### **Week 10: Abstract Classes and Interfaces**
- **Content:**
  - Abstract classes and abstract methods.
  - Interfaces and their implementation.
  - Differences between abstract classes and interfaces.

#### **Week 11: Exception Handling**
- **Content:**
  - Understanding exceptions and error handling.
  - Using `try`, `catch`, `finally` blocks.
  - Creating custom exceptions.

#### **Week 12: Collections Framework**
- **Content:**
  - Introduction to Java Collections: `ArrayList`, `LinkedList`, `HashMap`, `HashSet`.
  - Iterating through collections using loops and iterators.

#### **End of Month 3: **Monthly Exam**

---

### **Month 4: Project Development and Review**

#### **Week 13: File I/O in Java**
- **Content:**
  - Reading from and writing to files using `FileReader`, `FileWriter`, `BufferedReader`, and `BufferedWriter`.
  - Understanding serialization and deserialization.

#### **Week 14: Recap and Advanced Topics Introduction**
- **Content:**
  - Reviewing key OOP concepts learned throughout the semester.
  - Introduction to advanced topics such as Java Streams and Lambda expressions.
  - Planning for the final project.

#### **Week 15: Final Project Presentation and Semester Review**
- **Content:**
  - Students present their final projects demonstrating their understanding of Java and OOP.
  - Peer reviews and constructive feedback.
  - Comprehensive review of the semester's content.

#### **End of Month 4: **Final Semester Exam**
- **Exam Components:**
  - **Comprehensive Written Exam:** Covering all topics from the semester, including Java fundamentals, OOP concepts, exception handling, and collections.
  - **Final Project Evaluation:**
    - **Criteria:** Functionality, code quality, use of OOP principles.

---

## **Semester 2: Web Development with HTML, CSS, JavaScript, and Spring Boot**

### **Overview:**
- **Duration:** 15 weeks (4 months)
- **Frequency:** 1 class per week
- **Focus:** Front-end development using HTML, CSS, JavaScript, and back-end development with Spring Boot to create a simple website.
- **Assessments:** Monthly exams, hands-on tasks, and a final project with a comprehensive final exam.

---

### **Month 1: Introduction to Web Development**

#### **Week 1: Introduction to HTML**
- **Content:**
  - Understanding the structure of a web page.
  - Basic HTML tags: `<!DOCTYPE>`, `<html>`, `<head>`, `<body>`.
  - Creating headings, paragraphs, and lists.

#### **Week 2: HTML Elements and Attributes**
- **Content:**
  - Working with links (`<a>`), images (`<img>`), and tables (`<table>`, `<tr>`, `<td>`).
  - Form elements: `<form>`, `<input>`, `<textarea>`, `<button>`.
  - Using attributes to modify elements.

#### **Week 3: Introduction to CSS**
- **Content:**
  - Understanding CSS syntax and selectors.
  - Applying styles to HTML elements.
  - CSS properties: color, background, font, margin, padding.

#### **Week 4: Advanced CSS Techniques**
- **Content:**
  - Box model and layout.
  - Flexbox and Grid for responsive design.
  - CSS positioning: static, relative, absolute, fixed.

#### **End of Month 1: **Monthly Exam**

---

### **Month 2: JavaScript Fundamentals**

#### **Week 5: Introduction to JavaScript**
- **Content:**
  - Understanding JavaScript and its role in web development.
  - Embedding JavaScript in HTML using `<script>` tags.
  - Basic syntax, variables, and data types.

#### **Week 6: Control Structures and Functions in JavaScript**
- **Content:**
  - Conditional statements (`if`, `else`, `switch`).
  - Looping constructs (`for`, `while`).
  - Defining and invoking functions.

#### **Week 7: DOM Manipulation**
- **Content:**
  - Understanding the Document Object Model (DOM).
  - Selecting and modifying HTML elements using JavaScript.
  - Event handling (e.g., `click`, `mouseover`).

#### **Week 8: Advanced JavaScript Concepts**
- **Content:**
  - Arrays and objects in JavaScript.
  - Introduction to ES6 features: arrow functions, let/const, template literals.
  - Basic debugging techniques.

#### **End of Month 2: **Monthly Exam**

---

### **Month 3: Introduction to Spring Boot and Back-End Development**

#### **Week 9: Introduction to Spring Boot**
- **Content:**
  - Understanding the basics of Spring Boot.
  - Setting up a Spring Boot project using Spring Initializr.
  - Overview of Maven/Gradle build tools.
- **Additional Tasks:**
  - Create a simple Spring Boot application that runs on a local server.
  - Configure project dependencies using Maven.

#### **Week 10: Building RESTful APIs with Spring Boot**
- **Content:**
  - Understanding REST principles.
  - Creating REST controllers and defining endpoints.
  - Handling HTTP requests and responses.

#### **Week 11: Connecting Front-End with Back-End**
- **Content:**
  - Understanding client-server architecture.
  - Making AJAX requests from JavaScript to Spring Boot APIs.
  - Handling JSON data exchange.

#### **Week 12: Database Integration with Spring Boot**
- **Content:**
  - Introduction to databases and JDBC.
  - Setting up a nonrelational database (MongoDB).
  - Connecting Spring Boot to a database.

#### **End of Month 3: **Monthly Exam**
---

### **Month 4: Project Development and Final Assessment**

#### **Week 13: Project Planning and Design**
- **Content:**
  - Defining project requirements and scope.
  - Designing the website structure and database schema.
  - Setting up version control using Git.

#### **Week 14: Building the Front-End Interface**
- **Content:**
  - Developing the website's user interface using HTML, CSS, and JavaScript.
  - Implementing responsive design principles.
  - Enhancing UI with interactive JavaScript features.

#### **Week 15: Integrating Front-End with Spring Boot Back-End**
- **Content:**
  - Connecting front-end forms and components to back-end APIs.
  - Implementing CRUD functionalities on the website.
  - Testing and debugging the integrated application.

#### **End of Month 4: **Final Semester Exam**
- **Exam Components:**
  - **Comprehensive Written Exam:** Covering HTML, CSS, JavaScript, and Spring Boot concepts learned throughout the semester.
  - **Final Project Submission and Evaluation:**
    - **Project Requirements:** A simple, fully functional website incorporating front-end and back-end technologies.
    - **Evaluation Criteria:** Functionality, design, code quality, integration between front-end and back-end, and presentation.
    - **Presentation:** Students demonstrate their websites, explaining the technologies used and challenges overcome.
